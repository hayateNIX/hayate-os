# 🌪️ Hayate OS (疾風)

A lightweight, fast and minimal Linux distribution based on Debian + APT.

## ⚡ Philosophy

- Speed over complexity
- Minimal but powerful environment
- Fully reproducible system
- Keyboard-driven workflow

## 🧱 Base

- Debian GNU/Linux
- APT package manager
- Wayland + Hyprland

## 🖥️ Desktop

- Hyprland (tiling WM)
- Waybar
- Wofi
- Kitty terminal

## 📦 Goal

To build a fast, modular and modern Linux distribution inspired by Arch-style customization but based on Debian stability.

## 🚧 Status

Early development stage
